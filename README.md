# 🌐 JSP & Servlet Web Application Repository

This repository contains my **Java Web Development practice projects** built using **JSP (Java Server Pages) and Servlets**.  
The purpose of this repository is to understand **server-side web development using Java** and build dynamic web applications following the **MVC architecture**.

---

# 📚 About This Repository

This repository includes implementations and examples developed while learning **JSP and Servlet technologies**.  
It focuses on building **dynamic web applications** that interact with databases and process client requests.

Topics covered in this repository include:

- Java Servlets
- JSP (Java Server Pages)
- MVC Architecture
- Request and Response handling
- Session management
- Database connectivity using JDBC
- Dynamic web page generation

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository

```bash
git clone https://github.com/ChaitanyaBhatt147/JSP_Servlet.git
cd JSP_Servlet
```

---

## 2️⃣ Build the project

```bash
mvn clean install
```

---

## 3️⃣ Deploy the application

Deploy the project on **Apache Tomcat Server** using your IDE such as:

- IntelliJ IDEA
- Eclipse

---

# 🌐 Access the Application

After deployment, open your browser and visit:

```
http://localhost:8080/
```
